10

Principles

The rules that keep the system feeling like one product.

Neutrals first, color second

A surface is white. A border is a 1px hairline. Color only appears when it carries meaning — a status, a metric, a data series. Never for decoration.

One italic serif moment per heading

Instrument Serif italic is the signature accent — use it on one or two words inside a display heading. Never body copy, never more than twice per section.

Hairlines over shadows

Cards are separated by 1px borders, not drop shadows. Shadows appear only on floating elements (popovers, the hero dashboard mockup).

Number every feature

Major feature sections are labelled with monospace two-digit indices (01, 02, 03). It signals editorial rigor and echoes the dashboard’s data-forward voice.

Pills are the handshake

Every new section opens with an eyebrow pill (“About”, “Features”, “Testimonials”). It orients the reader before the headline lands.

Data cards are self-contained

Every metric includes label, value, and — where meaningful — a mono trend indicator. Never a label alone, never a value alone.
